# Kafka Cluster Management

Franz is the data layer that storage metadata about a fleet of Kafka. It give the reletionship between kafka and its resources, and its data structure helps to manage a big fleet with many kafkas cluster and many kafka topics and other resources that are highly distributed.


### Depedencies
- [Postgres](https://www.postgresql.org/)
- [Leinigen](https://leiningen.org/)
- [Clojure](https://clojure.org/) 
- [Plumatic Schema](https://github.com/plumatic/schema)
